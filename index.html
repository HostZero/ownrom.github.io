Install Instructions

  • Download latest OwnROM zip for your device
  • Download latest OwnDroid G-Apps
  • Move file to the download directory
  • Reboot into CWM/TWRP recovery
  • Wipe System, Data, Cache and Dalvik Cache (only wipe cache and dalvik if updating)
  • Flash ROM and G-Apps
  • Reboot And Enjoy!
  • Note: As always, no one but only you can be held responsible if you (soft) brick your device.
